Line Height Controls — How should it appear in the toolbar?

Pick the style that fits best alongside the existing Bold / Italic / H1 / H2 buttons.

A — Dropdown select

Compact, fits in one slot. Shows current line height. Values: 1, 1.5, 2, 2.5, 3.

B — Icon buttons (tight / normal / relaxed)

3 preset buttons with line-spacing icons. Active state highlighted. Quick to click, visual metaphor is clear.

C — Segmented number buttons

Pill-style group showing all values (1 / 1.5 / 2 / 2.5 / 3). Active value highlighted. All options visible at a glance.
